Smart Health Diagnosis is an AI-powered health assessment tool that uses machine learning to analyze symptoms and predict potential health conditions. It's designed to provide quick, preliminary health insights based on user-reported symptoms.

ü§ñ How It Works

The system uses a Random Forest classifier trained on medical symptom-disease data to make predictions. Here's the process:

  • User selects their current symptoms from a comprehensive list
  • The ML model analyzes the symptom pattern
  • The system returns the most likely condition with confidence score
  • Relevant precautions and recommendations are provided
